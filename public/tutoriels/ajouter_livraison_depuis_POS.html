ajouter la livraison depuis un point de vente

Ce guide vous montrera comment ajouter des options de livraison à partir d'un point de vente dans Estoriom.

Accédez à demo-pos.estoriom.com

1. Cliquer sur "Ventes"

2. Accéder au POS (PDV)

3. Séléctionner le / les produit(s)

4. Cliquer sur la section encadrée

5. Ouvrez la liste pour les options de livraison

Vous avez ajouté et configuré avec succès les options de livraison à partir d'un point de vente à Estoriom. Cette configuration permet une gestion efficace des services de livraison pour vos points de vente.
