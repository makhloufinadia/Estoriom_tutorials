Ajouter un paiement partiel au point de vente

Ce guide explique comment ajouter un paiement partiel dans le système de point de vente.

Accédez à demo-pos.estoriom.com

1. Cliquer sur "Ventes"

2. Cliquer sur "POS(PDV)"

3. Ajouter un produit au panier

4. Cliquer sur "Paiement multiple"

5. Enter Partial Payment Amount

6. Saisir le montant partiel

7. Cliquer sur le bouton pour ajouter un second paiement pour le reste

8. Cliquer ici pour finaliser le paiement

Vous avez ajouté avec succès un paiement partiel à une vente dans le système de point de vente. Ce processus permet une gestion flexible des paiements en répartissant les paiements entre plusieurs méthodes.
