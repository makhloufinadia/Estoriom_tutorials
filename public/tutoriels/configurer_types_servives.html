Configuration des types de services

Ce guide vous explique comment configurer les types de services dans Estoriom afin de rationaliser la gestion de vos services.

Accédez à demo-pos.estoriom.com

1. Aller dans Paramètres

2. Cliquer sur Types de services

3. Cliquer sur Ajouter

4. Nommer le servives clairement (ex: Livraison à domicile)

5. Définir le tarif standard

6. Sauvegarder

Vous avez configuré avec succès de nouveaux types de services dans Estoriom, y compris leur dénomination et la définition des frais d'emballage. Cette configuration vous aidera à gérer vos services plus efficacement et à personnaliser vos offres pour vos clients.
