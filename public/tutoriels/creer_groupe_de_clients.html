Créer un groupe de clients

Ce guide vous explique comment configurer des groupes de clients et gérer les contacts clients dans Estoriom.

Accédez à demo-pos.estoriom.com

1. Aller dans contact

2. Puis "Groupe de clients"

3. Cliquer sur "Ajouter"

4. Ajouter votre groupe (VIP, Gros, Revendeur etc)

5. Sauvegarder

6. Maintenant aller dans client

7. Cliquer sur Ajouter

8. Affecter le groupe comme indiqué

9. Sauvegarder

Vous avez configuré avec succès les groupes de clients et géré les filtres clients dans Estoriom. Cette configuration permet une segmentation efficace et un contrôle des prix pour vos contacts.
