Créer un combo de produits

Cette démonstration montre comment créer une combinaison de produits dans Estoriom.

Accédez à demo-pos.estoriom.com

1. Créer les produits simples d'abord (si pas existants)

2. Select SKU Field

3. Créer le deuxieme produit

4. Si vous voulez définir votre quantité d'alerte cocher cette case

5. La quantité d'alerte (ou stock d'alerte / stock minimum)

6. Pour créer le combo aller dans "Ajouter un produit" (comme toujours)

7. Nom du produit: "Kit souris + Clavier pro"

8. SKU: COMBO001

9. Type de produit : Combo (ce qui change tout)

10. Vous pouvez rechercher les produits (pour les combiner) par SKU, par nom ou en scannant le code-barre

11. Définissez la marge

12. Sauvegarder le produit combo

Cette démonstration montre comment créer efficacement une combinaison de produits dans Estoriom, en regroupant plusieurs articles dans un seul kit. Vous pouvez rationaliser la gestion des stocks et améliorer le suivi des ventes en utilisant cette fonctionnalité dès aujourd'hui.
